# What Is DocWeaver?

**DocWeaver** is a fictional, template-driven documentation system. It models a realistic workflow:
- **Reusable templates** (README, changelog, guides)
- **Central variables** (project name, version, author) via YAML
- **Linting hooks** for consistent Markdown

## Core ideas
1. **Clarity over cleverness**
2. **Consistency through structure**
3. **Sustainability by design**
