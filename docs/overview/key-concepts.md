# Key Concepts

- **Template**: A Markdown file with placeholders (e.g., `{{project_name}}`).
- **Variables**: Central values (YAML) injected into templates at generation time.
- **Profiles**: Optional sets of variables for different products/audiences.
- **Lint Rules**: Formatting checks for headings, code blocks, and links.

> DocWeaver favors **human-readable** Markdown and **minimal tooling**.
