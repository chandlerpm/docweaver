# Content Standards

- **Structure**: Intro → Prereqs → Steps → Verify → Next steps
- **Headings**: H1 per page; use H2/H3 hierarchically
- **Snippets**: Prefer short, copy-pastable blocks
- **Images**: Add alt text; include brief captions
- **Links**: Relative where possible; validate periodically

> See also: **Branching & Reviews** and **Changelog Guidelines** (stubs).
