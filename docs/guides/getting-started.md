# Getting Started

1. **Copy the examples**
   - Duplicate the files in `/examples` into your project.

2. **Set your variables**
   - Edit `examples/sample-config.yml` to match your project.

3. **Customize templates**
   - Open `examples/template-example.md` and replace placeholders.
   - Replace `{{project_name}}`, `{{version}}`, `{{author}}` manually for now.

4. **Preview**
   - Open the generated Markdown and check headings, links, and tone.

Next: **[Style Guide](style-guide.md)** for voice/formatting rules.
